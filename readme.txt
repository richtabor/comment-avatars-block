The Comment Avatars block dynamically displays your post's comment activity.

== Description ==

The Comment Avatars block dynamically displays your post's comment activity. Based on the comment status, it will:

* Display a friendly "Add comment" link when there are no comments.
* Show avatars of up to three recent commenters when comments exist.
* Display the total comment count to highlight discussion activity.

Perfect for blogs and community-driven websites where you want to highlight active discussions and encourage new participants to join the conversation.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly (if available).
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Add the block from the inserter within the WordPress editor.

== Frequently Asked Questions ==

= Does this work with Gravatar? =

Yes, the plugin uses WordPress's built-in avatar system, which includes Gravatar support.
